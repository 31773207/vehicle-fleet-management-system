## Backend Setup

1. Install PostgreSQL

2. Create database named: pfe_db

3. Make sure PostgreSQL is running (default port 5432)

4. Default credentials:

   * username: postgres
   * password: 123

5. Run Spring Boot backend

The database tables will be created automatically using JPA entities.

Server runs on:
http://localhost:8080

## Frontend Setup

1. Go to frontend folder
2. Run:

npm install
npm start

Frontend will connect to backend automatically.
Frontend will run on port 3000 or 3001
